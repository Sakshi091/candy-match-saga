# 🍬 Candy Match Saga

A super fun match-3 puzzle game like Candy Crush!  
Made with pure HTML + Canvas.

## 🎮 Play Now
→ **[Play the game here!](https://Sakshi091.github.io/candy-match-saga/)**

## How to Play
- Tap a candy, then tap an adjacent one to swap  
- Match 3 or more same candies  
- Reach the target score to win the level  
- Enjoy smooth animations, falling candies & celebration confetti!
